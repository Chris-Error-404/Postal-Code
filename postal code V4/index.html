What is a Postal Code?

A postal code is a series of letters or digits or both, sometimes including spaces or punctuation, included in a postal address for the purpose of sorting mail.

Postcode, postal code and zip code all represent the same thing. Every country have a preferred way of referring to it.

In Canada, it is popularly referred to as postal code; in the UK, Australia and New Zealand, postcode; while in the United States, it is mainly referred to as zip code.

The Nigerian Postal Service NIPOST is a government-owned and operated corporation responsible for providing postal services in Nigeria. Unlike other countries that use alphanumerical digits for their postal codes, in Nigeria, postal codes contain just 6 numerical digits.
